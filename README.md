# Sales Executive Performance Dashboard (Excel)

## Project Overview
This project is an interactive Sales Executive Performance Dashboard built using Microsoft Excel.
The dashboard analyzes sales performance, target achievement, and regional distribution of sales executives.

---

## Key Metrics
- Total Sales: 5063
- Target per Executive: 500
- Target Hit %: 55.24%
- Total Sales Executives: 141

---

## Features
- KPI Cards
- Top & Bottom Executive Analysis
- Target Hit vs Away From Target %
- Region-wise Performance
- Pivot Tables & Pivot Charts
- Interactive Dashboard
